# LuCI App Aurora Config

[English](README.md) | **简体中文**

## 简介

**LuCI App Aurora Config** 是专为 [luci-theme-aurora](https://github.com/eamonxg/luci-theme-aurora) 设计的配置应用，后续会增加更多个性化配置（头发已掉光.jpg）。

你可以通过该应用配置：

- **主题颜色**：内置 Color Picker，自由选择喜欢的配色
- **布局方式**：支持设置顶部导航栏悬浮时的二级菜单展示布局

---

## ⚠️ 注意事项

- 本应用仅支持 **luci-theme-aurora ≥ v0.8.9_beta**
- 使用低于此版本的主题将无法生效!!!
